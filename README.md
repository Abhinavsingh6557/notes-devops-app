# Notes Management App - DevOps Project

A simple Flask Notes Management application built to demonstrate an end-to-end DevOps workflow.

## Features

- Add notes
- View notes
- Edit notes
- Delete notes
- Search notes
- SQLite database
- Health check endpoint

## Tech Stack

- Python
- Flask
- SQLAlchemy
- SQLite
- HTML
- CSS

## DevOps Tools Planned

- Git & GitHub
- Pytest
- Docker
- Docker Compose
- GitHub Actions
- Ubuntu VM
- Nginx
- Prometheus
- Grafana

## Run Locally

```bash
pip install -r requirements.txt
python app.py
